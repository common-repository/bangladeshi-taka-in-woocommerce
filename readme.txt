This plugin adds Bangladeshi Taka (BDT) to WooCommerce powered store  

== Description ==

[Priced Host](http://www.pricedhost.com/) is proudly announcing our first plugin for wordpress users specially eCommerce site owners from Bangladesh. By default WooCommerce didn't support Bangladeshi currency (BDT); this little plugin adds the functionality to support BDT.

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Bangladeshi Taka (BDT) will be automatically added to WooCommerce > Setting
4. Press Save changes.
